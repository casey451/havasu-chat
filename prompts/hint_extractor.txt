- location: ONLY if the user mentions a specific location within Lake Havasu City (e.g. "near the island," "staying downtown," "by the channel"). Do not extract generic phrases like "right now," "at home," or references to cities other than Lake Havasu.

If the user does not mention either, set extracted_hints to null. Do not fabricate hints. When in doubt, return null.

Examples (value of extracted_hints only):
- User: "my 6-year-old likes BMX" → {"age": 6, "location": null}
- User: "we're staying near the channel" → {"age": null, "location": "near the channel"}
- User: "what's open right now" → null
- User: "good things for adults to do" → null
- User: "my kid wants something fun" → null

The user message will be sent in the next turn as plain text. Respond with JSON only, no markdown.
