# relay/

This directory holds chat-relay artifacts: HALT transcripts, sanity-check
outputs, status reports, and other ephemeral content Cursor produces for
the owner to forward to Claude during mobile-driven sessions.

Files in this directory are gitignored and are NOT part of project
history. They are working artifacts only.

If you are looking for project specs, methodology docs, or any
authoritative project artifact, those are in `docs/`. Anything in `relay/`
is ephemeral and may be deleted at any time without consequence.
